2048: Classic Puzzle

Rules and Info:
  • Press start to start playing.
  • The app needs minimum internet connection to work.
  • In this game you need to merge same numbers with each other until you get the number 2048, you can swipe right, left, top, down to move the boxes with numbers towards that side.
  • If all the boxes are filled and you can't merge any more numbers, then game over.
